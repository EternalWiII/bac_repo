# Генерація Javadoc-документації

Ця інструкція описує процес створення документації для Java-коду за допомогою Maven і Javadoc.

## Кроки

1. **Відкрийте термінал** у кореневій директорії проєкту (де знаходиться `pom.xml`).

2. **Виконайте команду для генерації Javadoc:**

   ```bash
   mvn javadoc:javadoc
   ```
3. Після успішного виконання команда згенерує документацію за шляхом:
   ```bash
   target/site/apidocs/
   ```
4. Для перегляду документації:
  - Відкрийте файл index.html у будь-якому браузері.
